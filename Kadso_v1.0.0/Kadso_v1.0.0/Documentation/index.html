Introduction

Kadso is a fully responsive, premium Bootstrap 5.3.3 Admin & Dashboard Template. Ideal for Web Apps, Dashboards, Admin Panels, and SaaS interfaces, Kadso features Gulp-based build tools, SCSS variables base, and RTL support. Quickly customize layouts and modes using data attributes without extensive code changes. Enjoy a beautiful UI with minimal coding effort.

Kadso has 185+ pages, 40+ components, 30+ charts, 20+ widgets with fully responsive design in Desktops, Tablets and Mobile devices.
